# Experimental Notebooks

이 폴더에는 탐색 및 추가 실험에 사용한 노트북을 보관합니다.

## `EDA.ipynb`

데이터 구조와 문제 유형을 먼저 파악하기 위한 노트북입니다.

확인한 내용:
- train/dev/test가 어떤 구조로 나뉘는지 확인
- dev에서 사람들이 같은 답을 골랐는지, 자주 갈렸는지 확인
- 질문을 개수, 재질, 색상, 문자처럼 이해하기 쉬운 범주로 나눠 확인
- 정답 분포와 보기 쏠림이 있는지 확인
- 이미지 resize, padding, crop이 개수·재질 문제에 어떤 영향을 줄지 후속 실험 후보로 확인

EDA는 점수를 바로 올리는 단계라기보다, 어떤 문제가 어려운지 먼저 확인하고 다음 실험 순서를 정하는 자료에 가깝습니다.
